Introdução

Seja bem-vindo ao Dev Quiz Fix. Sua privacidade é nossa prioridade absoluta. Este documento explica como lidamos com as informações no aplicativo.

Coleta de Dados

O Dev Quiz Fix foi projetado para ser um aplicativo focado na privacidade do usuário:

  • Não coletamos dados pessoais identificáveis (como nome, e-mail ou localização).
  • Não utilizamos contas de usuário em nuvem; todo o progresso é mantido apenas no seu aparelho.
  • Não transmitimos nenhum dado para servidores externos.

Armazenamento Local

O aplicativo utiliza o armazenamento interno do seu dispositivo (Shared Preferences) apenas para:

  • Salvar seu progresso em cada quiz.
  • Armazenar suas preferências de notificação para lembretes diários.

Esses dados permanecem no seu celular e são excluídos caso você desinstale o aplicativo ou limpe os dados nas configurações do sistema.

Notificações Locais

Se ativadas por você, as notificações são agendadas e disparadas localmente pelo seu sistema operacional. Não utilizamos serviços de Push externos ou rastreadores para este fim.

Serviços de Terceiros

Atualmente, o aplicativo não utiliza redes de anúncios ou ferramentas de análise externas (Analytics) que rastreiem seu comportamento ou dados.
